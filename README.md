# Yoga In Toulouse - GitHub Pages

## Comment déployer ce site en 2 minutes :
1. Crée un nouveau repository sur GitHub nommé `yogaintoulouse.github.io` (ou `yogaintoulouse`).
2. Ajoute tous les fichiers de ce dossier (`index.html`, `style.css`, `script.js`, `clubs/`) dans le repository.
3. Vas dans les **Settings** > **Pages** de ton repository.
4. Choisis la branche `main` et le dossier `/root`.
5. Ajoute ton domaine personnalisé `yogaintoulouse.fr` dans les paramètres GitHub Pages.
6. C'est en ligne !

## Stratégie SEO incluse :
- **Fichiers statiques rapides :** Pas de temps de chargement base de données (100/100 Google Speed).
- **Structure en Silo :** Une page d'accueil + des pages dédiées par club dans le dossier `/clubs/`.
- **Schema.org :** Chaque page de club contient des métadonnées JSON-LD pour que Google comprenne qu'il s'agit d'un business local (adresse, coordonnées GPS).
- **Mobile First :** Totalement responsive.
